

## JBoot 是什么

Jboot，专为大型分布式项目和微服务而生。

QQ交流群： 601440615

## maven 依赖

```xml
<dependency>
    <groupId>io.jboot</groupId>
    <artifactId>jboot</artifactId>
    <version>1.2.1</version>
</dependency>

```
**注意：** 很多些时候，Jboot版本已经更新，但是文档没有更新的情况下，请自行查看jboot的maven中央仓库最新版本。
